You can use your proxy or google proxy

1 sample of url using your proxy

	http://localhost/index.html#/true

	in This case you can set url proxy in file Controllers/feedCtrl.js change "http://localhost/proxy.php?url=" whatever you want!!!

2 - sample of url using google proxy

	http://localhost/index.html#/false

	the URL of google proxy is "https://ajax.googleapis.com/ajax/services/feed/load?v=1.0&num=25&callback=JSON_CALLBACK&q=";
